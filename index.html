Exercices d'algorithmie

Vous trouverez ici les solutions de quelques exercices effectués en javascript pour m'entraîner à l'algorithmie (lien vers page html et/ou fichier js sur mon github) :

  • Rot 13 : on convertit une phrase en une autre en d&calant de 13 lettres toutes les lettres de la phrase
    Page html Fichier js
  • Fizzbuzz : Affiche les nombres de 1 à 100.
    Si le chiffre est divisible par 3, remplace le chiffre par le mot « fizz ».
    Si le chiffre est divisible par 5, affiche le mot « buzz ».
    Si le chiffre est divisible par 3 et 5, affiche le mot « fizzbuzz ».
    Sinon fait simplement apparaitre le chiffre.
    Page html Fichier js
  • ParseToRoman : fonction appelée parseToRoman qui, pour un nombre donné, va retourner sa valeur en chiffres romains
    Page html Fichier js
  • Palindrome : dire si un mot est un palindrome ou non
    Page html Fichier js
  • Nombres hexadécimaux : Transformer des nombres hexadécimaux en nombres décimaux et additionner tous les nombres pairs entre eux
    Fichier js
  • Somme décimale :Transformer un nombre binaire en nombre décimal et faire la somme de tous les nombres impairs
    Fichier js
  • Nettoyer une phrase : Eliminer les nombres d'une phrase pour ne garder que les lettres
    Page html Fichier js
  • Isogramme : Dire si un mot donné est un isogramme ou non
    Un isogramme est un mot qui ne contient pas de lettre en double, consecutivement ou non.
    Exemples:
    isIsogram("aba") == false
    isIsogram("moOse") == false
    isIsogram("Dermato") == true
    Page html Fichier js
  • Bissextile : Déterminer si une année est Bissextile
    Page html Fichier js
  • Secondes en minutes : écrire une fonction pour convertir un nombre de secondes en nombre de minutes
    Page html Fichier js
